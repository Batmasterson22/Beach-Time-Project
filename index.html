Project Description:

Beach Time is an application that tracks consultants “on the beach”, or between clients. Users will register for the beach, upload their résumé, and tag their identities with useful metadata. Account executives will be able to search for available consultants based upon the provided metadata.

Project Assets:

This project will be created with ASP.Net MVC and Microsoft SQL Server.
